# Pensamento Computacional

Repositório da disciplina **Pensamento Computacional** — Curso de Ciência da Computação.

## 📚 Sobre a Disciplina

A disciplina de Pensamento Computacional apresenta 
os fundamentos do raciocínio lógico e da resolução de problemas por meio de conceitos computacionais, 
como decomposição, reconhecimento de padrões, abstração e algoritmos.
